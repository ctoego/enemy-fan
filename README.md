#### Enemy fan



Игра в которой вам нечего делать. Есть только вы и враги. Будьте бдительны и помните о своей цели.



#### Горячие клавиши

F8 - ограничение fps в 60
F7 - ограничение fps в 200
F6 - По разному

F2 - создание дополнительных зомби 2 уровня
F1 - Перемещение на 10 волну

Q - выстрел

#### Переменные

game - находимся ли мы в игре, даже если проиграли
GAME\_ZOMBIE - True, пока мы реально играем, если проиграем, становится False
GAME\_HEIGHT - высота игрового поля
SAVE - класс с сохранениями  и т.п.
DATA - файл конфигурации



#### Координаты

Представлен двумерным массивом nuppy, x и y
размер карты: 96 x 50 клеток(игровое поле меньше из за нижнего меню)
0 - пусто, только спрайт карты есть
1 - резерв
2 - турель
3 - шипы



#### спрайты

спрайты связанные с движением обновляются по своему внутреннему таймеру, их скорость не зависит от частоты кадров
пули летят со внутренней частотой 90 для плавного движения
